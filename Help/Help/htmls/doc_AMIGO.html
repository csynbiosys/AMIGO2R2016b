What's NEW

  • Advanced C based execution modes
  • openMP support to accelerate models with many experimental conditions

Compatibility

AMIGO is compatible with Linux and Windows. Windows users (32-bit and 64-bit OS version) must use a 32-bit Matlab installation.

Installation

To install basic functionality of AMIGO, run:

AMIGO_Startup;

The usage of Fortran or C models requires some installation steps. See :

  • Install Fortran utilities in AMIGO
  • Install C utilities in AMIGO

How to input problems in AMIGO

AMIGO is programmed making use of the so called Matlab structures. Structures are multidimensional Matlab arrays with elements called fields. These fields may be of any data type (arrays, matrices, strings of characters, etc.) and may be easily classified in subsets, therefore being quite comfortable for managing all input and output information. See: How to input problems in AMIGO.
